# Bozorp - Dark Circus Roleplay

This project was created by **Vexel Studios**.

## Overview
A premium web interface for the Bozorp Roleplay server, featuring a dark circus aesthetic with modern design elements and interactive components.

## Features
- **Immersive Hero Section**: Professional branding and server statistics.
- **Rules & Guidelines**: Compact, easy-to-read sections for server rules.
- **Activity Showcase**: Visual representation of server activities.
- **Mobile Responsive**: Fully optimized for all device sizes.
- **Vite-Powered**: Fast development and optimized production builds.

## Setup & Installation

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/)

### Development
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Deployment
To build the project for GitHub Pages:
1. Run the build command:
   ```bash
   npm run build
   ```
2. The production files will be generated in the `docs` folder.
3. Commit and push the `docs` folder to the `main` branch.
